# Food Farm 🌾🍎

A comprehensive desktop application designed to streamline farm management, track agricultural produce, and manage food distribution logistics. Built using **Java** and **JavaFX**, this system provides a clean, intuitive interface for farmers, suppliers, or administrators to oversee daily operations efficiently.

---

## 🚀 Features

*   **Dashboard Overview:** Real-time metrics on total crop yields, current inventory levels, and recent sales or distributions.
*   **Inventory & Crop Management:** Track crop types, planting dates, harvest schedules, and stock levels.
*   **Supplier & Buyer Directory:** Maintain records of suppliers (seeds, fertilizers) and distribution buyers with full contact logs.
*   **Order & Transaction Tracking:** Log sales, generate simple invoices, and monitor revenue streams.
*   **Modern UI Components:** A responsive, user-friendly interface powered by JavaFX controls and custom CSS styling.

---

## 🛠️ Tech Stack & Architecture

*   **Language:** Java (JDK 11 or higher recommended)
